# 소개

마법의 소라고둥처럼 단호하게 대답하는 디스코드 봇입니다.
<br>
랜덤 메시지부터 음성 호출, 음악 재생, AI 확장까지 지원 예정입니다.

# 기능

- **채팅 응답** : 사용자가 메시지를 입력하면 `돼.` / `안돼.` 등의 메시지 랜덤 응답
- **자동 채널 생성** : 서버에 초대되면 `똑똑해질 마법의 소라고둥` 전용 채널 자동 생성
